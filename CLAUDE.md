# Development Guidelines

## Versioning

Always increment the version number in `__manifest__.py` before committing any changes.

The version follows the format `17.0.X.Y.Z` where:
- `17.0` — Odoo major version (do not change)
- `X` — major feature or breaking change
- `Y` — minor feature or improvement
- `Z` — bug fix or small tweak

Example: `17.0.1.0.0` → `17.0.1.0.1` for a bug fix, or `17.0.1.1.0` for a new feature.
